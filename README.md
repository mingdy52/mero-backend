# mero-backend
Backend API for Mero - A travel diary app

**Why "Mero"?**  
Memory + Road = 여행길을 기록하는 여행 일기 앱

## 프로젝트 개요

여행 중 사용할 여행 일기 모바일 앱의 백엔드 API 서버입니다.

### 주요 기능
- 📝 여행 일기 작성 및 위치 기록 (GPS)
- 💰 경비 관리 및 환율 자동 환산
- 📸 사진 업로드 (최대 3장/일기)
- 🔄 오프라인 모드 지원 (SQLite ↔ MySQL 동기화)

## 기술 스택

### Backend
- Java 17
- Spring Boot 3.5
- MySQL 8.0
- JPA/Hibernate
- JWT Authentication
- Spring Security

### Infrastructure
- Railway (배포)
- AWS S3 (이미지 저장, 예정)
